# 🎮 Galz Games Quiz

Quiz ao vivo estilo Kahoot com avanço automático de perguntas.

## Como funciona

- **Perguntas avançam automaticamente** com contagem regressiva de 5 segundos após cada resposta
- **Resultados finais** só são exibidos quando o admin clicar em "Ver Resultados Finais"
- Jogadores veem o feedback (acerto/erro + pontos) e aguardam a próxima pergunta automaticamente

## Publicar no GitHub Pages

1. Crie um repositório público no github.com
2. Faça upload dos 3 arquivos: `index.html`, `style.css`, `app.js`
3. Settings → Pages → Branch: main → Save
4. Acesse: `https://SEU-USUARIO.github.io/galzgames`

## Login admin
- Usuário: `admin`
- Senha: `1234`

> ⚠️ Funciona entre abas do **mesmo navegador**. Para jogar entre dispositivos diferentes, é necessário um backend como Firebase.
