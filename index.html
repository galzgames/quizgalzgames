⚙️ Configurar Upload de Mídia

Dois serviços gratuitos, sem cartão de crédito:
📸 ImgBB — para fotos (imgbb.com)
🎵 Cloudinary — para áudios (cloudinary.com)
📸 ImgBB — Fotos
1. Acesse imgbb.com → Criar conta grátis
2. Vá em api.imgbb.com → copie a API Key
🎵 Cloudinary — Áudios
1. Acesse cloudinary.com → Free tier
2. Dashboard → copie o Cloud Name
3. Settings → Upload → Add upload preset → Mode: Unsigned → copie o nome
💡 Alternativa sem configuração: Cole links diretos de imagens já hospedadas (Google Drive, Imgur, etc.) no campo de link abaixo de cada foto/áudio.
